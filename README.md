# 🍲 The Ultimate Recipe Book

A fully responsive, interactive Recipe Book built to demonstrate core front-web development skills including DOM manipulation, event handling, and CSS Grid layouts. 

## 🌟 Features

* **Extensive Catalog:** Browse through 25 distinct recipes organized into intuitive categories (Starters, Main Course, Snacks, Dinner, and Desserts).
* **Dynamic Filtering:** Use the interactive navigation bar to instantly sort and view recipes by category without reloading the page.
* **Healthy Mode Toggle:** A custom switch that seamlessly reveals or hides detailed nutritional facts (Calories, Protein, Fat, Fibre, Sugars) for every dish.
* **Interactive Instructions:** Clickable "Cook This" buttons that expand to show step-by-step cooking instructions for each recipe.
* **Responsive Grid Design:** Clean, modern UI built with CSS Grid and smooth hover transitions for a polished user experience.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure and data embedding.
* **CSS3:** Custom styling, Grid layout, and interactive states (hover/active).
* **Vanilla JavaScript:** Explicit DOM manipulation, event listeners, and dynamic CSS styling (`style.display`).

## 🚀 How to Run

This project requires no build tools or external dependencies. 

1.  Clone or download this repository to your local machine.
2.  Ensure `index.html`, `style.css`, and `script.js` are in the same directory.
3.  Simply double-click `index.html` to open it in any modern web browser.
